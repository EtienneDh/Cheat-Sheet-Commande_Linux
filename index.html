Cheat Sheet: Commandes Linux

ls (list segment)

Permet de lister un répertoire.

-a: liste les fichiers cachés.
-l: Affichage detaillé du répertoire.

cd (change directory)

Permet de se promener dans le répertoire

cd -: Remonte au répertoire précedent
cd .. Remonte au répertoire parent

pwd Permet de savoir ou l'on se situe.

mv (move)

Permet de déplacer/renommer des fichers et répertoires.

-f: Ecrase les fichiers de destination sans confirmation
-i: Pareil, avec confirmation.

cp (copie) & rm (remove)

cp

Permet de copier des fichier et répertoires.

-r: Copie le répertoire et tout son contenu.
-v: Permet de suivre les copies en temps réel.

rm

Permet de supprimer des fichiers

-i: Demande confirmation avant d'effacer.
-r: Efface recursivement.

mkdir (make directory) & rmdir (remove directory)

mkdir

Permet de créer un répertoire vide

-p: Créer un répertoire parent s'il n'existe pas

rmdir

Supprime un répertoire

-p Supprime les répertoires parents s'ils deviennent vide

cat (concatenate) & more

cat

Affiche le contenu d'un fichier.

-n: Affiche les numéros de ligne.

more

Affiche le contenu d'un fichier page par page

-s Regroupe les lignes vides en une seule. -f Ne coupe pas les lignes longues.

find

Permet de chercher des fichiers et éventuellement d'effectuer des actions sur ceux-ci.

-name: Permet de faire une recherche par nom.
-type: Permet de faire une recherche par type.
-atime: Recherche par dernière date d'accés.
-user: Recherche par utilisateur.

Actions fréquentes

-exec: Effectue l'action demandée sur les fichiers trouvés. -ls Applique ls au fichiers trouvés.

grep (global regular expression print)

Recherche une chaine de caractère dans les fichiers

-c: Retourne le nombre de ligne au lieu des lignes elle-mêmes.
-i: Insensible à la casse.
-r: Recherche recursive.
